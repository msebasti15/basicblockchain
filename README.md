# Basic BlockChain  
  The objective of this project is to learn about BlockChain

  
## Run Locally  
Clone the project  

~~~bash  
  git clone https://github.com/msebasti15/basicblockchain.git
~~~

Go to the project directory  

~~~bash  
  cd basicblockchain
~~~

Install dependencies  

~~~bash  
go get github.com/davecgh/go-spew/spew
go get github.com/gorilla/mux
go get github.com/joho/godotenv
~~~

Start the server  

~~~bash  
go run ./cmd/app/main.go
~~~
